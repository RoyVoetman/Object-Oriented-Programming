# Object Oriented Programming
University assignments for Object Oriented Programming in Java.

## Subjects covered

### Week 1
* Multi-threading
* Java FX
* Event-driven programming

### Week 2
* Abstract classes
* Interfaces
* Strategy pattern
* Observer pattern
* Decorator pattern
* Factory pattern

### Week 3
* Singleton pattern
* Command pattern
* Adapters & facades
* Template method
* Iterator
* Composite pattern
* State pattern
* Proxy pattern
* Model View Controller (MVC)

### Week 4
* Patterns in dynamic languages
* Inversion of Control (IoC)
* Generics
* Handling exceptions and assertions
* File i/o
* Networking
* Serialization with JSON
* Database programming
* Profiling
